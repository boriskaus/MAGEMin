Python libraries		  
================

A serie of python libraries are needed to generate the C version of the database.

First update your system to python3 and use pip3 to install the following libraries:

.. code-block:: shell

	pip3 install numpy
	pip3 install sympy
	pip3 install scipy
	
C tools
=======

Valgrind (debugger)

.. code-block:: shell

	sudo apt-get install valgrind
